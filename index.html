Ad Campaign Performance Dashboard

This project aims to create an interactive Google Looker Studio report for analyzing advertising campaign performance using PostgreSQL data. It retrieves metrics on Ad Spend, CPC, CPM, CTR, and ROMI via a custom SQL query. The report includes three main charts: a combo chart for Ad Spend and ROMI trends, a line chart for monthly campaign activity, and a table with heatmaps to compare metrics across campaigns. Filters for campaign name and ad date enable customized views for deeper insights.
